Задание содержит Python-скрипт для поиска клиентов, которые не совершили ни одного заказа, на основе сопоставления таблиц пользователей и покупок.

### **Функционал:**

Скрипт выполняет следующие действия:
1. Принимает на вход два набора данных: `Customers` (id, name) и `Orders` (id, customerId).
2. Выполняет объединение таблиц методом `left merge` по идентификаторам клиентов.
3. Фильтрует данные, выявляя пользователей, у которых отсутствуют записи в таблице заказов (`isna()`).
4. Формирует итоговый отчет, содержащий только имена клиентов.
5. Сохраняет результат в файл `output/customers_without_orders.csv`.

### **Требования:**

1. Для запуска необходим Python 3 и библиотека `pandas`.

```bash
pip install pandas
```

### **Запуск:**

1. Подготовьте исходные данные в формате Pandas DataFrame (или загрузите их из CSV).
2. Запустите скрипт:

```bash
python Task_3.py
```

3. Результат (список клиентов) будет доступен в файле `output/customers_without_orders.csv`.
