Задание содержит Python-скрипт для статистического анализа и визуализации временных рядов измерений из CSV-файла.

Функционал:

Скрипт выполняет следующие действия:
1.Загружает данные из measurements.csv.
2.Рассчитывает для каждой серии (S1, S2, S3) количество записей (n), среднее значение, стандартное отклонение и корреляцию Пирсона.
3.Сортирует таблицу метрик по названию серии.
4.Сохраняет итоговые расчеты в output/metrics.csv с фиксированным порядком столбцов.
5.Создает линейный график динамики изменений по дням.
6.Создает столбчатую диаграмму для сравнения средних значений.
7.Сохраняет графики в высоком разрешении в папку output/figures/.

Требования:

1.Для запуска необходим Python 3 и библиотеки pandas, matplotlib, seaborn.

pip install pandas matplotlib seaborn

Запуск:

1.Разместите исходный файл measurements.csv в корневой директории проекта.
2.Запустите скрипт:

python Task_2.py

3.Результаты (таблица и графики) будут доступны в папке output/.
