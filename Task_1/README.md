Задание содержит Python-скрипт для обработки логов измерений из CSV-файла и генерации сводного отчета по партиям.

## Функционал
Скрипт выполняет следующие действия:
1. Загружает данные из `data/tech_screen_sample.csv`.
2. Исключает записи с отрицательными значениями в столбце `value`.
3. Удаляет дубликаты `record_id` внутри каждой партии (оставляя только первое вхождение).
4. Группирует данные по столбцу `batch`.
5. Рассчитывает количество записей (`count`) и сумму значений (`total`) для каждой партии.
6. Округляет итоговую сумму до 2 знаков после запятой.
7. Сортирует результат по названию партии.
8. Сохраняет результат в `output/result.csv`.

## Требования
Для запуска необходим Python 3 и библиотека `pandas`.

```bash
pip install pandas
```

## Запуск
1. Разместите исходный файл в директории `data/tech_screen_sample.csv`.
2. Запустите скрипт:
```bash
python Task_1.py
```
3. Результат будет доступен в папке `output/`.
